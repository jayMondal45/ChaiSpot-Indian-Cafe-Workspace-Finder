## ğŸ¯ Usage Guide

### For Users
1. **Browse Cafes**: Scroll through featured cafes on the homepage
2. **Search**: Use the search bar to find cafes by location
3. **Filter**: Click city tags to see cafes in specific cities
4. **View Details**: Each cafe card shows amenities, pricing, and seating
5. **Get Directions**: Click "View Map" to open Google Maps

### For Cafe Owners
1. Navigate to the "Add Cafe" section
2. Fill in all required fields:
   - Cafe name and location
   - Seating capacity (1-200)
   - Chai/Coffee price (â‚¹10-500)
   - Upload a cafe image URL
   - Google Maps URL for directions
3. Select amenities (WiFi, restrooms, power outlets, call-friendly)
4. Choose cafe type (Chai, Coffee, or Both)
5. Preview your listing in real-time
6. Submit to add to the directory
